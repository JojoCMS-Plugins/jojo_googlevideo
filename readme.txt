Google videos embed filter
--------------------------

Make it easy to embed google videos into page content.

To add a video to a page:
    Edit the page and add a piece of text like this:
    [[googlevideo:video url]]
    eg [[googlevideo:http://www.googlevideo.com/v/fYTJ9v2vsaE]]

Will accept the url as either the url from your browser
    eg [[googlevideo:http://video.google.com/videoplay?docid=-1534976813496691579]]
    or the embed url
    [[googlevideo:http://video.google.com/googleplayer.swf?docId=-1534976813496691579&hl=en]]
